## OpenScholar: Create a free site 
We're here to help university marketing, web & IT teams publish COVID-19 news and educational resources in real time. We have made OpenScholar websites free to you. Create your free site at https://go.theopenscholar.com/

With OpenScholar, it's easy to create a higher education website. It's free for university web, IT and marketing teams through June 30, 2020.

## OpenScholar: web site-creation package
* full-featured
* open-source
* designed for the academic community

For a non-technical overview, see the [User Documentation](http://docs.openscholar.harvard.edu)

For developers, see the [glossary](https://github.com/openscholar/openscholar/wiki/Glossary).
